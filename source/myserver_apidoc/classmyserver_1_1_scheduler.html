void myserver::Scheduler::start ( )

调度器开始运行

start 将初始化线程池，main 线程以外的所有线程将开始执行调度协程 为了主线程正确退出， main 线程不会在 start 方法中切换到调度协程

◆ stop()

void myserver::Scheduler::stop ( )

停止调度器

主线程将在 stop 方法中切换到调度协程真正开始任务调度，并在全部任务 调度结束后停止调度
