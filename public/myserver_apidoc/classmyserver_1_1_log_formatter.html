◆ LogFormatter()

myserver::LogFormatter::LogFormatter ( const std::string &  pattern)

构造一个日志格式化器

参数
pattern日志格式化器的模式串

模式串参数说明：

  • m 消息
  • p 日志等级
  • c 日志器名称
  • d 日期时间，后面可跟一对括号指定时间格式，比如%d{%Y-%m-%d %H:%M:%S}，这里的格式字符与 C 语言 strftime 一致
  • r 该日志器创建后的累计运行毫秒数
  • f 文件名
  • l 行号
  • t 线程 id
  • F 协程 id
  • N 线程名称
  • T 制表符
  • n 换行

默认格式："%d{%Y-%m-%d %H:%M:%S}%T%t: %N%T%F%T[%p]%T[%c]%T<%f:%l>%T%m%n" 默认格式说明：年-月-日 时-分-秒 线程id: 线程名称 协程id [日志等级] [日志器名称] <文件名:行号> 日志内容
